# Game

Podemos utilizar html e componentes React em arquivos MDX. Por exemplo, o componente `Game` é importado e renderizado abaixo. O jogo é uma versão simplificada do famoso jogo do dinossauro do Chrome. 

O mais legal de utilizar componentes React em arquivos MDX é que podemos criar componentes reutilizáveis e utilizá-los em diversos arquivos, sem a necessidade de repetir o código, o `Game` é um exemplo disso, importamos ele aqui e no arquivo `mdx-tests.md`.

Para jogar, clique no jogo ou utilize a tecla `Space` para pular sobre os obstáculos.
